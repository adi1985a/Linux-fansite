Popular Commands

Navigation

ls - list files cd - change directory pwd - show path mkdir - new directory rmdir - remove dir find - search files locate - find files tree - show structure

Files

cp - copy mv - move/rename rm - remove touch - new file cat - show content nano - text editor chmod - permissions chown - change owner

System

top - processes ps - process list kill - stop process df - disk space du - dir size free - memory info sudo - admin rights apt - packages

Network

ping - connection test ip a - IP address netstat - network stats ssh - remote access wget - download curl - transfer data nmap - port scan host - DNS lookup
