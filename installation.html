Installation Process

1. Preparation

Before installation, backup your data and download the ISO image of your chosen distribution.

2. Creating Bootable USB

Use programs like Rufus or Etcher to create a bootable USB with the system image.

3. Disk Partitioning

Prepare appropriate partitions for the system: root (/), swap, and home.

4. System Installation

Follow the installation wizard, selecting language, timezone, and other settings.
